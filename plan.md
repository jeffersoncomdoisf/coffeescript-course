### Why CoffeeScript?

* clean syntax
    * little or no need for {} () and ;
* added features
    * classes
	* array comprehensions
	* splats 
	* object enumeration
	* ...
* sins of the father (fixing javascript's faults)
    * removes ===
	* existential operator
	* fixes lexical scoping
